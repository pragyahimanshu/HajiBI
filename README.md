# 🎬 HajiBI Streaming Platform

> A modern movie discovery and entertainment platform built with React.js, TMDB API, and Firebase.

## 📖 Overview

HajiBI is a modern movie discovery platform designed to help users explore trending movies and TV shows through a clean, responsive, and engaging user interface. The platform allows users to search content, browse curated collections, manage personal watchlists, and receive personalized recommendations powered by the TMDB API.

The name **HajiBI** is inspired by **Hajipur, Bihar**, representing the vision of building modern digital products rooted in local identity with a global perspective.

---

## ✨ Features

- 🎬 Browse Trending Movies & TV Shows
- 🔍 Advanced Movie Search
- ⭐ Personalized Recommendations
- ❤️ Like & Save Favorite Movies
- 📚 My Watchlist
- 🎭 Curated Movie Collections
- 🎥 Movie Trailer Playback
- 🔐 Firebase Authentication
- 📱 Fully Responsive Design
- 🌙 Modern Dark UI

---

## 🛠 Tech Stack

- React.js
- Vite
- Firebase Authentication

- Axios
- Swiper.js
